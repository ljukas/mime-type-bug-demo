# MITT SKOLVAL FRONTEND
Enduser-facing frontend for collecting school choices.

## Getting started
```
yarn install
yarn start
```

## Git precommit-hook
```
└── bin
    └── hooks
        └── pre-commit
```
Place in .git folder to add hash extension to package.json version field on commit.
